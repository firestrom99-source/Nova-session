## 🔹 วิธีที่ 1: ใช้โปรแกรม Clone (แนะนำที่สุด ง่าย ไม่ต้องลงใหม่)

#### ขั้นตอนโดยสรุป:
1. เสียบ M.2 1TB ตัวใหม่เข้ากับเมนบอร์ด (สล็อต M.2)

3. เปิดโปรแกรม → เลือก “Clone this disk” บน SSD 256GB เดิม
4. เลือก M.2 1TB เป็นปลายทาง
5. เลือก Copy ทั้งพาร์ทิชัน (System, Recovery, EFI ฯลฯ)

6. เริ่ม Clone → รอจนจบ
7. เข้า BIOS → ตั้งค่า Boot จาก M.2 ตัวใหม่
8. ทดสอบว่า Windows บูตจาก M.2 ใหม่ได้แล้ว
9. เมื่อชัวร์แล้วจะฟอร์แมต SSD ตัวเก่าทิ้งหรือเก็บไว้ก็ได้

- **HDMI / DisplayPort** – ใช้งานต่อหน้าจอโดยตรงได้ หากใช้ CPU ที่รองรับกราฟิกในตัว เช่น Ryzen G-series
- **LAN (RJ‑45)** – ใช้สายแลน รองรับความเร็วสูงถึง 1 Gbps
- **Audio Jacks** –  
  - **สีฟ้า** = Line‑In (ต่ออุปกรณ์เสียงเข้า)  
  - **สีเขียว** = Line‑Out (ต่อลำโพงหรือหูฟัง)  
  - **สีชมพู** = Mic‑In (ต่อไมโครโฟน)

### 🎯 จุดเชื่อมโยง

- **iGPU** เช่น Ryzen 7 3700X ไม่มีกราฟิกในตัว — พอร์ต HDMI และ DisplayPort จะไม่มีสัญญาณ ถ้าใช้งานโดยไม่มีการ์ดจอแยก
- พอร์ต USB ครบครันทั้งชนิดความเร็วต่ำและสูง รวมถึงความเร็วสูงสุด 10Gbps สำหรับการใช้งานในอนาคต
- ไม่มีพอร์ต USB-C หรือ Wi‑Fi ในตัว — หากต้องการต้องเพิ่มการ์ดเสริม

## 🔍 **สาเหตุหลักที่ทำให้ Windows Update ช้า/ค้างนาน:**

### 1. **ขนาดของไฟล์อัพเดตใหญ่ (GB ขึ้นไป)**
- Windows 10/11 มักปล่อยอัพเดตใหญ่ (Feature Update) ปีละ 1–2 ครั้ง ซึ่งขนาดเกิน 2–4 GB
- ถ้าเน็ตช้า → โหลดช้า
- บางครั้งหลังโหลดเสร็จ ยังต้อง **แตกไฟล์ (Decompress)** และเช็คไฟล์อีกนาน (ขึ้นกับ SSD/HDD)

---

### 2. **Disk ช้า หรือใช้ HDD (แทน SSD)**
- ถ้าใช้ **HDD แบบเก่า** → Windows จะอ่าน/เขียนไฟล์อัพเดตช้ามาก โดยเฉพาะขั้นตอน "Installing Updates" หรือ "Applying Changes"  
- SSD เร็วกว่า HDD หลายเท่าตัว (คนใช้ SSD จะไม่เจอปัญหานี้หนักเท่าไหร่)

---

### 3. **มี Process หรือ Service รันเต็มเครื่อง**
- Antivirus, โปรแกรมเช็คระบบ, บริการใน Background เช่น OneDrive Sync ทำให้ CPU/Disk ไม่ว่าง → Windows Update ทำงานช้าลง

---

### 4. **ปัญหาเกี่ยวกับ Windows Update Cache**
- ไฟล์เก่า/ซ้ำในโฟลเดอร์ **SoftwareDistribution** ทำให้เกิดค้าง/วนลูป → บางทีต้องล้าง Cache นี้เพื่อแก้

---

### 5. **ปัญหาเกี่ยวกับ Driver / Hardware ไม่สมบูรณ์**
- Windows Update บางครั้งรอ Driver หรือระบบฮาร์ดแวร์ เช่น Chipset, SSD, VGA ที่ไม่รองรับ Update version ล่าสุด → ค้างนานกว่าปกติ

---

### 6. **ติดตั้ง "Cumulative Updates"**
- Microsoft รวม Patch หลายตัวในหนึ่งไฟล์ → ทำให้ต้องเช็คและติดตั้ง Patch เก่าไปด้วย (แม้ลง Windows ใหม่แล้ว)

---

## 🔧 วิธีแก้หรือช่วยให้อัพเดตเร็วขึ้น:

1. **ใช้ SSD แทน HDD** → ลดเวลาไปเกิน 50–80%
2. **เชื่อมต่ออินเตอร์เน็ตเสถียร** (ไม่ใช้ Hotspot หรือ Wi-Fi ที่หลุดบ่อย)
3. **ล้าง Windows Update Cache:**
   - เปิด Command Prompt (Admin)
   - พิมพ์:  
     ```
     net stop wuauserv
     net stop bits
     del /f /s /q %windir%\SoftwareDistribution\*
     net start wuauserv
     net start bits
     ```
4. **ปิด Antivirus ชั่วคราว (เฉพาะระหว่าง Update)**
5. **อย่าเปิดหลายโปรแกรมขณะอัปเดต**
6. **ถ้าเก่ามาก (Build เก่ามากๆ)** — บางครั้งต้อง "Clean Install" Windows ใหม่ถึงจะเร็วกว่า
