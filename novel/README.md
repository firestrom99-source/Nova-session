# NOVA Universe – Master Index

ยินดีต้อนรับสู่คลังเนื้อหาโลกนิยาย NOVA × Aetherion  
โฟลเดอร์นี้เป็นศูนย์กลางรวบรวมข้อมูลทั้งหมดของจักรวาล รวมถึงตัวละคร อาวุธ ศัตรู โครงเรื่องหลัก และข้อมูลลอว์ภายใน

## Directory
- **characters/** — โปรไฟล์ตัวละครหลัก–ตัวประกอบ พร้อมข้อมูลเชิงลึก
- **weapons/** — อาวุธ เทคโนโลยี สิ่งประดิษฐ์ พลังพิเศษ
- **enemies/** — บอส ฝ่ายตรงข้าม มอนสเตอร์ องค์กรศัตรู
- **locations/** — สถานที่ เมือง มิติ ระบบต่างดาว
- **plot-main/** — เส้นเรื่องหลัก จุดหักมุม เหตุการณ์สำคัญ
- **plot-side/** — เควสเสริม เนื้อเรื่องรอง ขยายจักรวาล
- **lore/** — ประวัติศาสตร์ ความเชื่อ โครงสร้างจักรวาล
- **blueprint/** — NOVA Blueprint, ระบบพลังงาน, เมตาดาต้า, วิธีใช้งานทีม NOVA
