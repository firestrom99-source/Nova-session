How this archive works
Step 0 · Manual mode

Minimal workflow for saving Nova × Aetherion logs into this vault. Everything lives in a single private repository; this page is only a read-only index.

Basic workflow
  • Export / copy a chat from ChatGPT.
  • Save it locally as YYYY-MM-DD_topic-or-tag.md or .txt.
  • Upload the file into this repository (web upload is fine).
  • The filename is parsed and appears in the Session index list automatically.
File naming convention
  • Suggested pattern: YYYY-MM-DD_topic-or-tag.md
  • Example: 2025-12-09_monitor-shopping.md
  • If the name doesn’t follow this pattern (e.g. nova ep.1 การระบุนอนอันตราย.txt) it still appears — just without a parsed date/tag.

Best practice: use English slugs with _ or - between words for nicer formatting.
